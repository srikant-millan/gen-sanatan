Learning Path

Courses for Students

6 - 8 Weeks each

Data Science Foundations

150 Instructor led hours

Total 5 Courses

Data is everywhere today - from retail websites to smart devices, to internet, social medias, bioinformatics, finances etc. This massive growth in the amount and variety of data needs the data scientists to make sense of these data and uncover the hidden solutions to messy business problems. This learning path consists of five courses that introduces you to Data Science from a practitioner point of view. You get to know about the different kinds of data and the database management system. You get the practical knowledge on how to write the queries to process data and do the data analysis in a technical way and derive inferences from it. The courses also provide the knowledge of different machine learning algorithms.

Data Analytics

150 Instructor led hours

Total 5 Courses

Learn how to code in Python for Data Science for Data Analysis and Visualization. You get to know about the different kinds of data and the database management system. You get practical knowledge on how to write the SQL queries and do the data analysis in a technical way and derice inferences from it. This is an action-packed learning path for Data Analysts or aspiring Data Analysts. You get to know different Python Packages for Data Analysis and Visualization like numpy, scipy, matplotlib, seaborn etc. You will get to know how to create beautiful and meaningful graphs/plots and telling a story about your data.

Machine Learning

150 Instructor led hours

Total 5 Courses

Throughout the guided lectures and labs, you will get hands on experience on how to deal with data problems. You will start with introduction to data science, core concepts, inferential statics. Then, you will start with the basic machine learning concepts and algorithms using standard Python libraries like scikit-learn to solve the real life data problems. In the advanced courses, you will learn the working principles of the algorithms and build your own library.

Deep Learning

150 Instructor led hours

Total 5 Courses

In this learning path, you will start with the baic concepts of data science and inferential statistics. You will learn the concepts of Deep Learning process and the Python libraries like TensorFlow. Then you will get the hands on experience in solving problems using Deep Learning methods. Throughout the course, you will be able to see how python libraries can be used in curve fitting, regression, classification and minimizing the error functions. You will get to know different types of deep architectures like RNN, CNN and Autoecoders.

Courses for Professionals

2 - 16 Hours each
Data Science for Professionals

2 Instructor led hours

An overview of Data Science methods and techniques

This is the age of Information and Data. We believe that every professional needs to be a data literate. This is a two hours program designed for individual contributors, junior to mid-level professionals coming from either private or public sectors seeking to manage or will manage data science teams or vendors. You will get to know the overview of data processing techniques, some mathematical and statistical terms and tools used in a data science project and the principles and practices of data science in industry. We will demonstrate few hands-on example how inferences are drawn from the data to make decisions.

Data Science for Managers

2 Instructor led hours

An overview of how data-based decisions can drive your business

This is a non-technical and non-mathematical overview of data science designed for project managers, product managers, Research and Development managers and CEOs. This is a detailed overview mainly to explain how you can make better business decions for your company using the inferences drawn from the historical data. We will demonstrate different tools and techniques used for data analysis. You will see several examples and stories explainign dat driven business decisions. Also, get familiar with beautiful and meaningful graphs and plots for your business reporting hod how effective they are.

Data Science 2 - Workshop

16 Instructor led hours

Two days workshop on data and familiarity with ML algorithms

If your team or company is starting or planning to start your own Data Analysis or Data Science project but not sure how to start or where to start. What are the processes and tools and artifacts produced in a Data based project. This course is for you. This is a two days workshop onsite at your worksite or online depending on your requirement, we disccuss the different data analysis tools and techniques, data mining processes. We discuss how to run a project from start to end (taking it to production and produce value out of it.)

This is a hands-on workshop where the participants get to write code, execute it to solve real life problems.
